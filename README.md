# TypeScript

This project is part of the Holberton School Web React curriculum. It introduces the fundamentals of TypeScript, including interfaces, classes, functions, namespaces, declaration merging, ambient namespaces, advanced types, and nominal typing.

## Learning Objectives

By completing this project, you will learn:

* Basic TypeScript types
* Interfaces and Classes
* Functions and TypeScript typing
* DOM manipulation with TypeScript
* Advanced types
* Namespaces
* Declaration merging
* Ambient namespaces
* Nominal typing using branding

## Tasks

### Task 0 - Student Interface

* Create a Student interface
* Create student objects
* Render student information in a table using TypeScript and DOM manipulation

### Task 1 - Interfaces and Classes

* Create Teacher interface
* Extend Teacher with Directors interface
* Create typed functions
* Create StudentClass with interfaces

### Task 2 - Advanced Types

* Create Director and Teacher interfaces
* Implement Director and Teacher classes
* Use type predicates
* Implement string literal types

### Task 3 - Ambient Namespaces

* Create custom type definitions
* Work with external JavaScript libraries
* Use declaration files (.d.ts)

### Task 4 - Namespaces & Declaration Merging

* Create namespaces
* Extend interfaces using declaration merging
* Create Subject, Cpp, React, and Java classes

### Task 5 - Nominal Typing

* Create branded interfaces
* Implement MajorCredits and MinorCredits
* Use nominal typing patterns in TypeScript

## Installation

Clone the repository:

```bash
git clone https://github.com/Randa-hb10/holbertonschool-web_react.git
cd holbertonschool-web_react
```

Navigate to any task directory:

```bash
cd TypeScript/task_0
```

Install dependencies:

```bash
npm install
```

Build the project:

```bash
npm run build
```

Run the development server:

```bash
npm run start-dev
```
